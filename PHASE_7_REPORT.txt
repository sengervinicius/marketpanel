================================================================================
TECHNICAL DETAILS
================================================================================

Touch Events:
  - onTouchStart: Record initial touch position
  - onTouchMove: Track movement for swipe detection
  - onTouchEnd: Execute action with e.preventDefault()
  - No 300ms tap delay

CSS Patterns:
  - touch-action: pan-y (allow vertical scroll, intercept horizontal)
  - max-height transitions for smooth expand/collapse
  - GPU acceleration for animations
  - Mobile-first responsive design

Design Tokens Used:
  - --row-height-touch: 44px
  - --space-*: 4px, 8px, 12px, 16px spacing
  - --duration-fast: 200ms animations
  - --ease-default: cubic-bezier(0.4, 0, 0.2, 1)
